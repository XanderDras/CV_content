# Current Project
This repository is a centralized, modular system for managing my CVs, cover letters, and professional project descriptions. It is designed to allow rapid, high-quality adaptation of my application materials for different companies, roles, and languages (English and Portuguese).

## What we are building
*   A master template system using LaTeX for professional, ATS-friendly formatting.
*   A repository of modular content (experiences, skills, projects) that can be easily swapped in or out depending on the target role.
*   A streamlined workflow where a target job description can be fed into the system to output a highly customized CV and cover letter.

## What good looks like
*   **Clean Code:** LaTeX files are cleanly indented, well-commented, and logically separated.
*   **Impact-Driven:** Every bullet point on the CV clearly demonstrates value, using the X-Y-Z formula (Accomplished [X] as measured by [Y], by doing [Z]).
*   **Tailored:** The final output directly mirrors the keywords and core requirements of the target job description.
*   **Organized:** The directory structure remains strict. English assets stay in `/english`, Portuguese assets in `/portuguese`, and project drafts in `/Projects`.

## What to avoid
*   **One-Size-Fits-All:** Avoid sending a generic, bloated CV to every company.
*   **Formatting Errors:** Avoid LaTeX syntax errors, overflowing text, or mismatched fonts/margins.
*   **Clutter:** Avoid hoarding outdated or irrelevant older experiences in the main tailored output. Keep the master files comprehensive, but the final exported PDFs ruthlessly concise.
